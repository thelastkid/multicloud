# MLOps Multi-Cloud Project

## Overview
This project implements a multi-cloud MLOps system for deploying and managing machine learning models.

## Modules

### Backend (Policy Engine)
Handles decision-making and deployment logic.

### ML Model
Handles model training and inference.

### Deployment
Handles containerization and Kubernetes deployment.

### Monitoring
Handles system monitoring and visualization.
